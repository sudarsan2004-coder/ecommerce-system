# E-Commerce System

A full-stack e-commerce web application developed using  
HTML, CSS, JavaScript, Java, Spring Boot, and MySQL.

## Features
- User registration & login
- Product listing
- Add to cart
- Order management
- Admin product management

## Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Java, Spring Boot
- Database: MySQL
- Tools: Git, GitHub

## How to Run the Project

### Backend
1. Open project in Eclipse / IntelliJ
2. Update MySQL credentials in `application.properties`
3. Run as **Spring Boot App**
4. Server runs on `http://localhost:8080`

### Frontend
- Open `index.html` in browser
